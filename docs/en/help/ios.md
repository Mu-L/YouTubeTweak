# Enable the Safari Extension

Step 1, click the extension icon on the right side of the address bar

<img src="../../img/help/ios-steps/enable-1.png" alt="Enable-Step-1" width=400/>

Step 2, click `Manage Extensions`

<img src="../../img/help/ios-steps/enable-2.png" alt="Enable-Step-2" width=400/>

Step 3, set the `YouTube Tweak` extension to enabled

<img src="../../img/help/ios-steps/enable-3.png" alt="Enable-Step-3" width=400/>

<a name="safari-setting"></a>

# Use and Configure the Extension on the YouTube Website

Open [YouTube.com](https://www.youtube.com), then click the extension icon in the address bar.  
Click `YouTube Tweak`. When you have not granted access to YouTube, the icon is gray and has an exclamation mark.

<img src="../../img/help/ios-steps/setting-1-2.png" alt="Setting-Step-1-2" width=400/>

In the confirmation dialog that appears, click `Always Allow`

<img src="../../img/help/ios-steps/setting-3.png" alt="Setting-Step-3" width=400/>

After that, this extension can work normally. Click the extension icon in the address bar again to enter the configuration screen.

<img src="../../img/help/ios-steps/setting-4.png" alt="Setting-Step-4" width=400/>
