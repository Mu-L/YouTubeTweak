# 功能说明

[中文](/docs/zh-cn/FUNCTIONS.md) | [English](/FUNCTIONS.md) | [日本語](/docs/ja/FUNCTIONS.md)

YouTube Tweak 是一个用于增强 YouTube 网页版体验的浏览器扩展。本文按照扩展弹出面板里的分类顺序整理，重点说明每个功能有什么用、开启后会发生什么、适合什么场景，以及需要注意的限制。

本扩展主要在桌面版 YouTube 网页中生效。部分功能只在视频播放页、首页、搜索页、评论区或 Shorts 相关区域生效，具体范围会在对应功能里说明。

<a name="player"></a>

# 播放器

这一类功能用于增强视频播放页的观看体验，主要作用在 YouTube 播放器和视频播放页。

<a name="player-quality"></a>

## 锁定画质

<a name="player-quality-auto-switch"></a>

### 自动切换视频画质

开启后，扩展会在每次进入或切换视频时，尝试把视频画质切换到你选择的清晰度。

适合的使用场景：

- 希望每个视频都尽量使用固定清晰度。
- 不想每次手动打开 YouTube 设置菜单调整画质。
- 网络环境稳定，想优先使用 1080p、1440p、4K 或更高画质。

可配置选项：

- 开关：控制是否自动切换视频画质。
- 目标画质：可选择 8K、4K、1440p、1080p、720p、480p、360p、240p 或 144p。

注意事项：

- 如果当前视频没有你选择的画质，扩展会自动降到这个画质以下可用的最高画质。
- 该功能只影响视频播放页。
- YouTube 本身可用的画质会受视频源、设备、网络和账号状态影响。

<a name="player-speed-button"></a>

## 倍速按钮

<a name="player-speed-button-show"></a>

### 显示倍速按钮和倍速滑块

开启后，播放器控制栏里会显示自定义倍速按钮，用户可以直接点击按钮切换播放速度，不需要打开 YouTube 原本的设置菜单。

适合的使用场景：

- 经常在 1.5x、2x 等固定倍速之间切换。
- 看教程、直播回放、长视频时需要快速调速。
- 想把常用倍速放在播放器上直接点。

可配置选项：

- 显示倍速按钮/倍速滑块：控制播放器里是否显示扩展的倍速入口。
- 启用的按钮：可选择 0.25x、0.5x、0.75x、1x、1.25x、1.5x、1.75x、2x、2.25x、2.5x、2.75x、3x、5x、10x。
- 启用倍速滑块：开启后，可以按住倍速区域左右拖动，连续调整倍速。
- 鼠标滚轮步长：滚轮调速可以跟随已启用的倍速按钮，也可以使用 0.0625、0.125、0.25、0.5、1 或自定义步长。

注意事项：

- 倍速滑块依赖“显示倍速按钮/倍速滑块”开启，并且至少需要启用两个倍速按钮。
- 鼠标放在倍速区域滚动滚轮时，会直接调整播放速度。
- 自定义滚轮步长支持很细的数值，但过小会让调整变慢，过大会让速度跳变明显。

<a name="player-speed-button-save"></a>

### 记住播放速度

开启后，扩展会记住你使用过的播放速度，并在之后的视频中自动恢复。

适合的使用场景：

- 总是用同一个倍速看大多数视频。
- 不同频道想使用不同倍速。
- 经常在学习类、娱乐类、音乐类频道之间切换。

可配置选项：

- 记住播放器速度：保存一个通用播放速度。
- 为每个频道分别记住播放速度：按频道保存播放速度。

注意事项：

- 同时开启通用记忆和分频道记忆时，分频道记忆优先。
- 分频道记忆需要扩展能识别当前视频所属频道。
- 记忆数据可以在通用设置里选择随设置一起导出。

<a name="player-volume-booster"></a>

## 音量增强器

<a name="player-volume-booster-gain"></a>

### 在播放器中增加真实音量增益

开启后，扩展可以在 YouTube 自带 100% 音量之上继续提高声音，并在播放器里显示一个音量增强按钮。

适合的使用场景：

- 视频本身声音太小。
- 系统音量已经调高，但 YouTube 仍然不够响。
- 想临时增强某个视频的音量，而不是调整整个系统。

可配置选项：

- 在倍速栏旁显示增强控制：控制播放器里是否显示音量增强按钮。
- 默认启用音量增强：进入视频时默认开启增强。
- 增强级别：可选择 1.25x 到 5x 的增益级别，每档间隔 0.25x。

使用方式：

- 点击播放器内的音量增强按钮，可以即时开关增强。
- 鼠标放在按钮上滚动滚轮，可以提高或降低增强级别。

注意事项：

- 声音增强过高可能导致破音、噪声或音质变差。
- 部分浏览器可能要求用户先和页面交互后，声音增强才会真正开始工作。
- 如果隐藏播放器内按钮，默认启用仍然可以生效，但不能在播放器里临时点击切换。

<a name="player-video-zoom"></a>

## 画面缩放

<a name="player-video-zoom-wheel"></a>

### 用鼠标滚轮缩放视频画面

开启后，可以在视频画面上滚动鼠标滚轮来放大或缩小画面。放大后，可以拖动画面来移动观看区域。

适合的使用场景：

- 视频里有小字、小细节或远处画面需要看清。
- 想放大横屏视频中的局部区域。
- 看教学、录屏、监控画面时需要临时观察细节。

使用方式：

- 鼠标移到视频画面中间区域，向上滚动放大。
- 放大后向下滚动缩小，直到恢复原大小。
- 放大后按住鼠标左键拖动，可以移动画面。

注意事项：

- 最大可放大到 4 倍。
- 鼠标在播放器按钮、控制栏、链接、输入框等区域上时，不会触发画面缩放。
- 该功能只作用于当前视频画面，不改变视频源文件。

<a name="player-subtitle"></a>

## 字幕

<a name="player-subtitle-save-status"></a>

### 记住字幕启用状态

开启后，扩展会记录你是否打开了字幕，并在后续视频中尝试恢复字幕状态。

适合的使用场景：

- 常看外语视频，希望字幕默认保持打开。
- 部分频道需要字幕，部分频道不需要字幕。
- 不想每个视频都手动开关字幕。

可配置选项：

- 记住字幕启用状态：记录通用字幕状态。
- 为每个频道分别记住字幕启用状态：按频道记录字幕状态。

注意事项：

- 自动恢复字幕状态需要同时开启“记住字幕启用状态”和“为每个频道分别记住字幕启用状态”。
- 如果视频没有字幕，扩展无法强行生成 YouTube 字幕。
- 分频道记忆需要扩展能识别当前视频所属频道。

<a name="player-function-buttons"></a>

## 功能按钮

<a name="player-function-buttons-rotate"></a>

### 旋转视频按钮

开启后，播放器里会出现旋转按钮。每点击一次，视频画面旋转 90 度。

适合的使用场景：

- 视频方向上传错误。
- 想横屏观看竖屏内容，或临时调整观看方向。

注意事项：

- 旋转只影响当前页面里的显示效果，不会修改视频本身。
- 旋转 90 度或 270 度时，扩展会尽量缩放画面以适配播放器区域。

<a name="player-function-buttons-mirror"></a>

### 镜像视频按钮

开启后，播放器里会出现镜像按钮。点击后可以水平翻转视频画面，再次点击可恢复。

适合的使用场景：

- 跟练舞蹈、健身、乐器动作时，希望左右方向更符合自己视角。
- 观看被镜像上传的视频时临时恢复方向。

注意事项：

- 镜像只影响显示效果，不会改变视频文件。
- 镜像可以和旋转同时使用。

<a name="player-function-buttons-screenshot"></a>

### 视频截图按钮

开启后，播放器里会出现截图按钮。点击后会截取当前视频帧，并在新窗口中打开图片。

适合的使用场景：

- 保存视频中的某一帧画面。
- 做笔记、记录教程步骤或保存画面参考。

使用方式：

- 点击截图按钮后，扩展会打开一个新窗口显示截图。
- 在截图窗口中按 Ctrl+S 可以保存图片。

注意事项：

- 浏览器弹窗拦截可能会阻止截图窗口打开。
- 如果当前视频帧还没有准备好，截图可能失败。
- 截图文件名会尽量使用视频标题和当前播放时间。

<a name="player-button-display"></a>

## 按钮显示方式

<a name="player-button-display-native"></a>

### 控制播放器原生按钮显示

该功能可以控制 YouTube 播放器中部分原生按钮的显示方式，让播放器控制栏更清爽。

适合的使用场景：

- 不使用某些 YouTube 原生按钮，希望减少干扰。
- 想强制显示某些被 YouTube 隐藏的按钮。
- 想让播放器控制栏更符合个人习惯。

可配置按钮：

- 自动播放按钮。
- 字幕按钮。
- 设置按钮。
- 迷你播放器按钮。
- 画中画按钮。
- 剧场模式按钮。
- 在电视上播放按钮。
- 全屏按钮。

可配置显示方式：

- 自动：保持 YouTube 默认显示方式。
- 隐藏：隐藏对应按钮。
- 强制启用：尽量让对应按钮保持可见。

注意事项：

- 字幕按钮只支持“自动”和“隐藏”，不提供“强制启用”。
- 如果 YouTube 页面结构变化，个别按钮的显示控制可能暂时失效。

<a name="player-mini-player"></a>

## 滚动迷你播放器

<a name="player-mini-player-scroll"></a>

### 滚动时让视频悬浮播放

开启后，在视频播放页向下滚动离开播放器时，当前视频会变成悬浮迷你播放器，方便一边看视频一边浏览评论或推荐内容。

适合的使用场景：

- 看视频时想同时浏览评论区。
- 想在播放页往下翻内容，但不想让视频离开视线。
- 长视频播放时需要边看边找信息。

可配置选项：

- 大小：可选择 360×203、420×236、480×270、560×315、640×360、720×405。
- 位置：可选择左上、顶部居中、右上、左下、底部居中、右下。
- 屏幕边距：设置迷你播放器离屏幕边缘的距离。
- 滚动超过播放器后激活：设置离开原播放器多少像素后进入迷你播放器。

注意事项：

- 该功能只在普通视频播放页生效。
- 全屏播放或画中画播放时不会进入迷你播放器。
- 迷你播放器上有关闭按钮，点击后本次滚动中会关闭悬浮；当你滚回播放器附近或切换视频后，会重新允许触发。
- 屏幕太小时，实际显示尺寸会自动缩小以避免超出屏幕。
- 小尺寸迷你播放器会隐藏部分不适合显示的控制按钮。

<a name="player-ui"></a>

## 界面

<a name="player-ui-ending-overlay"></a>

### 半透明结尾推荐内容

开启后，视频结尾覆盖在画面上的推荐视频、作者入口等元素会变成半透明，减少遮挡。

适合的使用场景：

- 经常被视频结尾推荐卡片挡住画面。
- 想看清视频最后几秒的内容。

注意事项：

- 鼠标悬停到这些推荐元素上时，它们会恢复正常显示，方便点击。
- 该功能只影响播放器中的结尾覆盖元素。

<a name="player-ui-progress-bar"></a>

### 视频下方进度条

开启后，当 YouTube 播放器控制栏自动隐藏时，视频底部会显示一条简洁进度条。

适合的使用场景：

- 想在不显示完整控制栏的情况下看到播放进度。
- 想减少 YouTube 原生进度条和控制栏的遮挡。

可配置选项：

- 是否显示进度条。
- 进度条高度，支持 1 到 20 像素。

注意事项：

- 进度条主要在播放器控制栏隐藏时显示。
- 直播视频不会显示这条进度条。

<a name="player-ui-time-tag"></a>

### 角落视频时间

开启后，当播放器控制栏自动隐藏时，播放器角落会显示当前播放时间和总时长。

适合的使用场景：

- 想随时知道播放到哪里，但不想打开完整控制栏。
- 看长视频时需要更明显的时间提示。

可配置选项：

- 是否显示视频时间。
- 字体大小，支持 8 到 48。
- 显示位置：左下、右下、左上、右上。
- 距离边缘的偏移，支持 0 到 200 像素。

注意事项：

- 时间角标主要在播放器控制栏隐藏时显示。
- 直播视频通常只显示当前播放时间。

<a name="player-other"></a>

## 其他

<a name="player-other-max-volume"></a>

### 最大音量时使用真正 100% 音量

开启后，当 YouTube 音量条处于 100% 时，扩展会确保视频元素本身也处于真正的满音量状态。

适合的使用场景：

- YouTube 显示音量已满，但实际声音仍偏小。
- 想确保播放器没有被额外压低音量。

注意事项：

- 该功能只在 YouTube 音量条为 100% 时介入。
- 这不是额外放大音量；如果需要超过 100% 的音量，请使用“音量增强器”。
- 生效时音量滑块会用醒目的颜色提示。

<a name="player-other-nonstop"></a>

### 禁用自动暂停视频

开启后，扩展会尽量阻止 YouTube 长时间播放后的自动暂停提示，并在检测到“视频已暂停，继续播放？”这类弹窗时自动处理。

适合的使用场景：

- 长时间播放音乐、播客、直播回放或学习视频。
- 不想 YouTube 因为长时间没有操作而暂停。

注意事项：

- 该功能只针对 YouTube 自动暂停类行为。
- 如果视频因为用户手动暂停、网络问题或页面限制而停止播放，扩展不会保证继续播放。

<a name="translate"></a>

# 翻译

这一类功能用于字幕和评论翻译。字幕翻译和评论翻译会使用在线翻译服务，因此需要网络可用。

<a name="translate-settings"></a>

## 翻译设定

<a name="translate-settings-target-language"></a>

### 翻译目标语言

该设置决定字幕翻译和评论翻译要翻译成哪种语言。

可配置选项：

- 跟随 YouTube 语言：自动使用当前 YouTube 页面语言。
- 指定语言：从列表中选择固定目标语言。

注意事项：

- 如果原文语言和目标语言相同，部分翻译功能会跳过自动翻译。
- 更改目标语言后，已显示的旧翻译内容可能需要刷新页面或重新加载评论后才会更新。

<a name="translate-settings-never-translate"></a>

### 始终不翻译的语言

该设置用于避免自动翻译某些语言的评论。

适合的使用场景：

- 你能直接阅读某些语言，不希望它们被自动翻译。
- 想避免中文评论、英文评论或其它熟悉语言重复出现译文。

注意事项：

- 这个设置主要影响评论自动翻译。
- 当评论被识别为不需要翻译的语言时，扩展会保留一个手动翻译按钮，用户仍可点击查看译文。

<a name="translate-subtitle"></a>

## 字幕翻译

<a name="translate-subtitle-enable"></a>

### 开启字幕翻译

开启后，扩展会在 YouTube 加载字幕时翻译字幕内容，并把译文显示到播放器字幕里。

适合的使用场景：

- 看外语视频时需要字幕翻译。
- 想保留原字幕，同时看到译文。
- 想直接把字幕替换成目标语言。

可配置选项：

- 开启字幕翻译：控制是否翻译视频字幕。
- 翻译方式：可选择“双语翻译”或“仅显示译文”。

翻译方式说明：

- 双语翻译：在字幕中同时显示译文和原文。
- 仅显示译文：只显示翻译后的内容。

注意事项：

- 该功能需要视频本身有可用字幕。
- 开启或关闭字幕翻译可能会刷新当前 YouTube 页面。
- 字幕内容过长或翻译服务失败时，扩展会保留原字幕。
- 字幕翻译属于会调整页面加载内容的功能，若页面出现异常，可先关闭该功能排查。

<a name="translate-comment"></a>

## 评论翻译

<a name="translate-comment-content"></a>

### 翻译评论内容

开启后，扩展会自动翻译评论区中的评论，并在评论原文附近显示译文。

适合的使用场景：

- 浏览多语言评论区。
- 看海外视频时想快速理解评论内容。
- 不想逐条点击 YouTube 原生翻译按钮。

可配置选项：

- 翻译评论内容：控制是否自动翻译评论。
- 翻译目标语言：决定评论翻译成哪种语言。
- 始终不翻译的语言：避免熟悉语言自动显示译文。

注意事项：

- 如果评论为空、只有符号，或被识别为不需要翻译，扩展会显示手动翻译按钮。
- 评论排序、加载更多评论或 YouTube 动态刷新评论区后，扩展会继续处理新出现的评论。
- 如果翻译服务不可用，评论会保持原样。

<a name="translate-comment-line-by-line"></a>

### 译文逐行显示在原文下方

开启后，对于包含多行内容的评论，扩展会尽量把每行译文放到对应原文下方，而不是把整段译文堆在评论末尾。

适合的使用场景：

- 翻译歌词、分段说明、长评论或多行时间轴。
- 希望原文和译文能逐行对照阅读。

注意事项：

- 该功能依赖“翻译评论内容”开启。
- 只有评论本身包含换行时，逐行显示才明显；普通单行评论仍会按普通方式显示译文。

<a name="other"></a>

# 其他

这一类功能用于调整 YouTube 页面布局、评论显示、广告干扰、Shorts、品牌外观、旧版布局和自定义样式。

<a name="other-home"></a>

## 首页

<a name="other-home-video-per-row"></a>

### 锁定首页推荐视频每行数量

开启后，可以固定 YouTube 首页推荐视频每行显示的数量。

适合的使用场景：

- 想让首页卡片更密集或更宽松。
- 不喜欢 YouTube 自动根据窗口宽度调整每行数量。
- 大屏幕上想显示更多视频。

可配置选项：

- 开关：控制是否锁定每行数量。
- 数量：输入每行显示的视频数量。

注意事项：

- 该功能主要影响首页推荐视频网格。
- 当前输入范围建议为 1 到 10；超出范围会被重置为 4。
- 页面宽度过窄时，YouTube 自身布局仍可能影响实际显示效果。

<a name="other-comment"></a>

## 评论

<a name="other-comment-nickname"></a>

### 显示评论者昵称

开启后，扩展会尝试显示评论者的频道名称，并把原本的账号名作为辅助信息显示。

适合的使用场景：

- 想看评论者更完整的频道显示名。
- 不习惯只看 YouTube 的账号名或句柄。

注意事项：

- 开启或关闭后可能需要刷新 YouTube 页面。
- 该功能需要访问评论者的频道页面来读取名称，因此加载速度取决于网络和 YouTube 页面响应。
- 如果频道信息无法读取，评论显示会保持 YouTube 原样。

<a name="other-comment-auto-more"></a>

### 自动展开长评论

开启后，长评论会自动展开，评论里的“显示更多”和“收起”按钮会被隐藏。

适合的使用场景：

- 经常阅读长评论或歌词评论。
- 不想反复点击“显示更多”。

注意事项：

- 该功能主要影响评论区。
- 开启后长评论默认展开，手动收起入口会被隐藏。

<a name="other-anti-ad"></a>

## 广告屏蔽

<a name="other-anti-ad-image"></a>

### 屏蔽图文广告

开启后，扩展会隐藏 YouTube 页面中的图文广告、广告位、部分 Premium 推广和页面广告模块。

适合的使用场景：

- 想减少首页、播放页和侧栏中的广告干扰。
- 想让推荐内容区域更干净。

注意事项：

- 这是实验功能，不保证能屏蔽所有广告。
- YouTube 页面结构变化后，部分广告可能重新出现。

<a name="other-anti-ad-video"></a>

### 屏蔽视频广告

开启后，扩展会尝试减少播放器里的视频广告，并在检测到广告视频时尽快跳过。

适合的使用场景：

- 想减少播放视频前后或中途插入的视频广告。
- 想让视频播放更连贯。

注意事项：

- 开启或关闭该功能可能会刷新当前 YouTube 页面。
- 这是实验功能，可能造成页面异常，也可能被 YouTube 检测。
- 部分情况下仍然需要手动点击 YouTube 的“跳过”按钮。
- 如果 YouTube 出现播放异常，可优先关闭该功能排查。

<a name="other-anti-ad-merch"></a>

### 屏蔽频道主商品推荐

开启后，扩展会隐藏频道主商品推荐、购物推荐栏、票务推荐和播放器中的部分商品推荐入口。

适合的使用场景：

- 不想在播放页看到商品货架或购物推广。
- 想减少视频周边商业推荐。

注意事项：

- 该功能只隐藏相关页面元素，不改变视频内容本身。
- YouTube 页面结构变化后，个别商品模块可能暂时无法隐藏。

<a name="other-shorts"></a>

## Shorts（短视频）

<a name="other-shorts-home-search"></a>

### 屏蔽首页和搜索结果中的 Shorts

开启后，首页推荐中的 Shorts 区块会被隐藏，搜索结果中的 Shorts 区块也会一起隐藏。

适合的使用场景：

- 不想在首页看到 Shorts 推荐。
- 搜索内容时只想看普通视频结果。

注意事项：

- 该功能影响首页和搜索页中的 Shorts 推荐区域。
- 它不会删除 YouTube 上的 Shorts 内容，只是隐藏对应入口或区块。

<a name="other-shorts-watch"></a>

### 屏蔽视频播放页中的 Shorts 推荐

开启后，视频播放页侧栏、简介区域或相关推荐区域中的 Shorts 推荐会被隐藏。

适合的使用场景：

- 看普通视频时不想被 Shorts 推荐打断。
- 想让播放页推荐区更专注于普通视频。

<a name="other-shorts-menu"></a>

### 在侧边栏里屏蔽 Shorts

开启后，YouTube 侧边栏、迷你侧边栏以及移动网页版底部导航中的 Shorts 入口会被隐藏。

适合的使用场景：

- 不使用 Shorts，希望减少入口干扰。
- 想避免误点 Shorts。

注意事项：

- 该功能主要隐藏入口，不影响直接打开 Shorts 链接。
- YouTube 不同语言或布局下，个别入口可能无法隐藏。

<a name="other-rollback"></a>

## 回退

<a name="other-rollback-player-layout"></a>

### 回到旧版播放界面布局

开启后，扩展会把 YouTube 播放页的一部分布局调整得更接近旧版样式，例如恢复较窄的右侧推荐栏和较小的推荐缩略图。

适合的使用场景：

- 不喜欢新版播放页右侧推荐过宽或缩略图过大。
- 希望播放页布局更紧凑。
- 遇到非 16:9 视频在播放器区域中显示不协调。

注意事项：

- 该功能主要影响桌面版视频播放页的双栏布局。
- 它是对页面外观的回退调整，YouTube 页面改版后效果可能变化。

<a name="other-appearance"></a>

## 外观

<a name="other-appearance-premium-logo"></a>

### 伪装 Premium Logo

开启后，左上角的普通 YouTube Logo 会显示为 YouTube Premium Logo。

适合的使用场景：

- 更喜欢 YouTube Premium Logo 的样式。
- 希望页面顶栏看起来像 Premium 账号。

注意事项：

- 该功能只改变 Logo 外观，不会开通 YouTube Premium、改变账号会员状态或解锁任何 Premium 权益。
- 开启或关闭该功能时，当前 YouTube 页面会自动刷新。
- YouTube 调整页面数据结构后，Logo 可能暂时无法替换。

<a name="other-custom-css"></a>

## 自定义 CSS

<a name="other-custom-css-inject"></a>

### 在 YouTube 注入自定义 CSS

开启后，可以把自己写的 CSS 样式应用到 YouTube 页面，用来调整颜色、隐藏元素或修改布局细节。

适合的使用场景：

- 你熟悉 CSS，想按个人习惯微调 YouTube。
- 想临时隐藏某些页面元素。
- 想做更深度的个性化外观调整。

可配置选项：

- 开关：控制是否启用自定义样式。
- 文本框：填写要应用到 YouTube 的 CSS。

注意事项：

- 文本框失去焦点后会保存内容，并应用到已打开的 YouTube 页面。
- 写错 CSS 可能导致 YouTube 页面显示异常。
- 关闭开关后，自定义样式会从页面中移除。

<a name="general"></a>

# 通用

这一类功能用于扩展本身的语言、设置、更新、关于信息和安装提示。

<a name="general-about"></a>

## 关于

<a name="general-about-info"></a>

### 查看扩展信息和项目入口

通用页面会显示扩展名称、版本号、构建时间，以及项目主页、更新日志和问题反馈入口。

适合的使用场景：

- 想确认当前安装的版本。
- 想查看更新内容。
- 想反馈问题或查看项目源码。

<a name="general-about-check-update"></a>

### 手动检查更新

点击版本号可以手动检查扩展是否有新版本。

注意事项：

- 如果浏览器不支持手动检查更新，扩展会提示你去扩展商店查看。
- 如果检查过于频繁，浏览器可能会暂时限制再次检查。

<a name="general-language"></a>

## 语言

<a name="general-language-switch"></a>

### 切换扩展显示语言

可以在通用页面切换扩展弹出面板的语言。

适合的使用场景：

- 想使用自己熟悉的语言查看设置。
- 想帮助检查某个翻译是否准确。

注意事项：

- 语言切换会立即影响扩展面板。
- 页面中提供翻译贡献者入口，方便查看或参与翻译。

<a name="general-config"></a>

## 插件设置

<a name="general-config-export"></a>

### 导出设置

可以把当前扩展设置导出为文本，方便备份、迁移到其他浏览器，或分享给其他用户。

可配置选项：

- 同时导出记忆：导出分频道倍速、字幕状态等记忆数据。

注意事项：

- 如果不勾选“同时导出记忆”，只会导出普通设置。
- 导出的内容会复制到剪贴板；如果复制失败，可以手动选择文本复制。

<a name="general-config-import"></a>

### 导入设置

可以粘贴之前导出的设置文本，将设置恢复到当前浏览器。

注意事项：

- 导入失败时，通常是文本格式不正确或内容不完整。
- 如果导入内容包含记忆数据，面板会要求你额外确认是否导入这些记忆。
- 导入记忆可能覆盖当前浏览器已有的分频道倍速、字幕等记忆。

<a name="general-config-reset"></a>

### 重置设置

可以把扩展设置恢复为默认值。

注意事项：

- 重置前会弹出确认提示。
- 重置后面板会回到“播放器”页面。
- 重置普通设置不会专门清空分频道记忆；如果要迁移或覆盖记忆，请使用导入导出里的记忆选项。

<a name="general-update-notice"></a>

## 更新和页面提示

<a name="general-update-notice-version"></a>

### 新版本提示

当浏览器检测到扩展有新版本时，通用页面会显示新版本提示。

可用操作：

- 查看更新日志。
- 立即更新。

注意事项：

- 正常情况下，关闭所有 YouTube 标签后浏览器会自动完成扩展更新。
- 点击“立即更新”会让扩展重新加载，并在更新完成后刷新已打开的 YouTube 标签页。

<a name="general-update-notice-reload-page"></a>

### YouTube 页面需要重新加载提示

当扩展更新后，已经打开的 YouTube 页面可能还在使用旧状态。此时页面上可能出现提示，要求重新加载页面。

适合的使用场景：

- 扩展刚更新，但 YouTube 页面里的功能看起来异常。
- 页面提示扩展已更新，需要重新加载。

注意事项：

- 点击提示里的重新加载按钮即可刷新当前页面。
- 重新加载后，页面会使用新的扩展状态。

<a name="general-update-notice-compatibility"></a>

### 兼容性警告

当开启字幕翻译或屏蔽视频广告时，通用页面会显示警告提示。

原因说明：

- 这两类功能会在 YouTube 加载字幕或广告相关内容时进行调整。
- 这种调整可能因为 YouTube 改版、网络问题或浏览器限制导致页面异常。

建议：

- 如果遇到视频无法播放、字幕异常、页面加载异常等问题，先关闭字幕翻译和屏蔽视频广告进行排查。

<a name="general-installed"></a>

## 安装欢迎页

<a name="general-installed-first-run"></a>

### 首次安装提示

首次安装扩展后，会打开一个欢迎页面，提示扩展已安装成功，并引导用户点击浏览器右上角扩展图标开始使用。

适合的使用场景：

- 刚安装扩展，需要知道从哪里打开设置面板。
- 想确认扩展已经安装完成。
