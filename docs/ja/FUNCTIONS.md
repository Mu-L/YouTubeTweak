# 機能説明

[中文](/docs/zh-cn/FUNCTIONS.md) | [English](/FUNCTIONS.md) | [日本語](/docs/ja/FUNCTIONS.md)

YouTube Tweak は、YouTube ウェブ版の体験を強化するためのブラウザー拡張です。本書は拡張機能のポップアップパネル内の分類順に整理し、各機能が何をするのか、有効化後に何が起きるのか、どのような場面に適しているのか、注意すべき制限は何かを重点的に説明します。

本拡張は主にデスクトップ版 YouTube ウェブで動作します。一部の機能は動画再生ページ、ホーム、検索ページ、コメント欄、または Shorts 関連領域でのみ動作します。具体的な範囲は各機能の説明で示します。

<a name="player"></a>

# プレーヤー

この分類の機能は動画再生ページの視聴体験を強化するためのもので、主に YouTube プレーヤーと動画再生ページに作用します。

<a name="player-quality"></a>

## 画質固定

<a name="player-quality-auto-switch"></a>

### 動画画質の自動切り替え

有効にすると、拡張機能は動画に入るたび、または動画を切り替えるたびに、選択した画質へ切り替えようとします。

適した使用場面:

- すべての動画でできるだけ固定画質を使用したい。
- 毎回 YouTube の設定メニューを手動で開いて画質を調整したくない。
- ネットワーク環境が安定しており、1080p、1440p、4K、またはそれ以上の画質を優先したい。

設定可能な項目:

- スイッチ: 動画画質を自動切り替えするかどうかを制御します。
- 目標画質: 8K、4K、1440p、1080p、720p、480p、360p、240p、または 144p を選択できます。

注意事項:

- 現在の動画に選択した画質がない場合、拡張機能はその画質以下で利用可能な最高画質へ自動的に下げます。
- この機能は動画再生ページにのみ影響します。
- YouTube 自体で利用可能な画質は、動画ソース、デバイス、ネットワーク、アカウント状態の影響を受けます。

<a name="player-speed-button"></a>

## 倍速ボタン

<a name="player-speed-button-show"></a>

### 倍速ボタンと倍速スライダーを表示

有効にすると、プレーヤー操作バーにカスタム倍速ボタンが表示されます。ユーザーは YouTube 標準の設定メニューを開かずに、ボタンを直接クリックして再生速度を切り替えられます。

適した使用場面:

- 1.5x、2x などの固定倍速を頻繁に切り替える。
- チュートリアル、ライブ配信のアーカイブ、長い動画を見るときに素早く速度調整したい。
- よく使う倍速をプレーヤー上に直接置きたい。

設定可能な項目:

- 倍速ボタン/倍速スライダーを表示: プレーヤー内に拡張機能の倍速入口を表示するかどうかを制御します。
- 有効なボタン: 0.25x、0.5x、0.75x、1x、1.25x、1.5x、1.75x、2x、2.25x、2.5x、2.75x、3x、5x、10x を選択できます。
- 倍速スライダーを有効化: 有効にすると、倍速エリアを押したまま左右にドラッグして、連続的に倍速を調整できます。
- マウスホイールステップ: ホイールによる速度調整は、有効化された倍速ボタンに従うことも、0.0625、0.125、0.25、0.5、1、またはカスタムステップを使用することもできます。

注意事項:

- 倍速スライダーは「倍速ボタン/倍速スライダーを表示」が有効であることに依存し、少なくとも 2 つの倍速ボタンを有効にする必要があります。
- マウスを倍速エリアに置いてホイールをスクロールすると、再生速度が直接調整されます。
- カスタムホイールステップは非常に細かい数値に対応しますが、小さすぎると調整が遅くなり、大きすぎると速度の跳ねが目立ちます。

<a name="player-speed-button-save"></a>

### 再生速度を記憶

有効にすると、拡張機能は使用した再生速度を記憶し、以後の動画で自動的に復元します。

適した使用場面:

- ほとんどの動画を常に同じ倍速で見る。
- チャンネルごとに異なる倍速を使いたい。
- 学習系、娯楽系、音楽系チャンネルを頻繁に切り替える。

設定可能な項目:

- プレーヤー速度を記憶: 共通の再生速度を 1 つ保存します。
- チャンネルごとに再生速度を個別に記憶: チャンネル別に再生速度を保存します。

注意事項:

- 共通記憶とチャンネル別記憶を同時に有効にした場合、チャンネル別記憶が優先されます。
- チャンネル別記憶には、拡張機能が現在の動画の所属チャンネルを識別できる必要があります。
- 記憶データは一般設定で設定と一緒にエクスポートできます。

<a name="player-volume-booster"></a>

## 音量ブースター

<a name="player-volume-booster-gain"></a>

### プレーヤー内で実際の音量ゲインを追加

有効にすると、拡張機能は YouTube 標準の 100% 音量を超えて音量を上げ、プレーヤー内に音量ブースターボタンを表示できます。

適した使用場面:

- 動画自体の音量が小さすぎる。
- システム音量を上げていても、YouTube の音がまだ足りない。
- システム全体ではなく、特定の動画だけ一時的に音量を強化したい。

設定可能な項目:

- 倍速バーの横にブースト操作を表示: プレーヤー内に音量ブースターボタンを表示するかどうかを制御します。
- デフォルトで音量ブーストを有効化: 動画に入ったときにデフォルトでブーストを有効にします。
- ブーストレベル: 1.25x から 5x までのゲインレベルを選択でき、各段階は 0.25x 間隔です。

使用方法:

- プレーヤー内の音量ブースターボタンをクリックすると、ブーストを即時にオン/オフできます。
- マウスをボタン上に置いてホイールをスクロールすると、ブーストレベルを上げ下げできます。

注意事項:

- 音量ブーストが高すぎると、音割れ、ノイズ、音質低下が起きる可能性があります。
- 一部のブラウザーでは、ユーザーが先にページと操作してからでないと、音量ブーストが実際に動作しない場合があります。
- プレーヤー内ボタンを非表示にしてもデフォルト有効は機能しますが、プレーヤー内で一時的にクリックして切り替えることはできません。

<a name="player-video-zoom"></a>

## 画面ズーム

<a name="player-video-zoom-wheel"></a>

### マウスホイールで動画画面をズーム

有効にすると、動画画面上でマウスホイールをスクロールして拡大または縮小できます。拡大後は、画面をドラッグして表示領域を移動できます。

適した使用場面:

- 動画内の小さい文字、細部、遠くの画面をはっきり見たい。
- 横向き動画の一部領域を拡大したい。
- 教学動画、画面録画、監視映像を見るときに一時的に細部を確認したい。

使用方法:

- マウスを動画画面の中央付近へ移動し、上にスクロールして拡大します。
- 拡大後は下にスクロールして縮小し、元のサイズに戻します。
- 拡大後にマウス左ボタンを押したままドラッグすると、画面を移動できます。

注意事項:

- 最大 4 倍まで拡大できます。
- マウスがプレーヤーボタン、操作バー、リンク、入力欄などの領域上にあるときは、画面ズームは発動しません。
- この機能は現在の動画画面にのみ作用し、動画ソースファイルは変更しません。

<a name="player-subtitle"></a>

## 字幕

<a name="player-subtitle-save-status"></a>

### 字幕の有効状態を記憶

有効にすると、拡張機能は字幕をオンにしているかどうかを記録し、以後の動画で字幕状態の復元を試みます。

適した使用場面:

- 外国語動画をよく見ており、字幕をデフォルトでオンにしておきたい。
- 一部のチャンネルでは字幕が必要で、一部のチャンネルでは不要。
- 動画ごとに字幕を手動でオン/オフしたくない。

設定可能な項目:

- 字幕の有効状態を記憶: 共通の字幕状態を記録します。
- チャンネルごとに字幕の有効状態を個別に記憶: チャンネル別に字幕状態を記録します。

注意事項:

- 字幕状態の自動復元には、「字幕の有効状態を記憶」と「チャンネルごとに字幕の有効状態を個別に記憶」の両方を有効にする必要があります。
- 動画に字幕がない場合、拡張機能は YouTube 字幕を強制生成できません。
- チャンネル別記憶には、拡張機能が現在の動画の所属チャンネルを識別できる必要があります。

<a name="player-function-buttons"></a>

## 機能ボタン

<a name="player-function-buttons-rotate"></a>

### 動画回転ボタン

有効にすると、プレーヤー内に回転ボタンが表示されます。クリックするたびに、動画画面が 90 度回転します。

適した使用場面:

- 動画の向きが誤ってアップロードされている。
- 縦向きコンテンツを横向きで見たい、または一時的に視聴方向を調整したい。

注意事項:

- 回転は現在のページ内の表示効果にのみ影響し、動画自体は変更しません。
- 90 度または 270 度回転した場合、拡張機能はプレーヤー領域に合うように可能な限り画面を拡大縮小します。

<a name="player-function-buttons-mirror"></a>

### 動画ミラーボタン

有効にすると、プレーヤー内にミラーボタンが表示されます。クリックすると動画画面を水平反転し、もう一度クリックすると元に戻ります。

適した使用場面:

- ダンス、フィットネス、楽器の動きを真似るときに、左右方向を自分の視点に合わせたい。
- ミラー状態でアップロードされた動画を一時的に正しい向きへ戻したい。

注意事項:

- ミラーは表示効果にのみ影響し、動画ファイルは変更しません。
- ミラーは回転と同時に使用できます。

<a name="player-function-buttons-screenshot"></a>

### 動画スクリーンショットボタン

有効にすると、プレーヤー内にスクリーンショットボタンが表示されます。クリックすると現在の動画フレームをキャプチャし、新しいウィンドウで画像を開きます。

適した使用場面:

- 動画内の特定のフレームを保存したい。
- メモ、チュートリアル手順の記録、画面参考の保存に使いたい。

使用方法:

- スクリーンショットボタンをクリックすると、拡張機能が新しいウィンドウを開いてスクリーンショットを表示します。
- スクリーンショットウィンドウで Ctrl+S を押すと画像を保存できます。

注意事項:

- ブラウザーのポップアップブロックにより、スクリーンショットウィンドウが開けない場合があります。
- 現在の動画フレームがまだ準備できていない場合、スクリーンショットに失敗することがあります。
- スクリーンショットのファイル名には、できるだけ動画タイトルと現在の再生時間を使用します。

<a name="player-button-display"></a>

## ボタン表示方式

<a name="player-button-display-native"></a>

### プレーヤー標準ボタンの表示を制御

この機能は YouTube プレーヤー内の一部の標準ボタンの表示方式を制御し、プレーヤー操作バーをよりすっきりさせます。

適した使用場面:

- 特定の YouTube 標準ボタンを使わないため、干渉を減らしたい。
- YouTube によって隠されている一部のボタンを強制的に表示したい。
- プレーヤー操作バーを自分の習慣により合わせたい。

設定可能なボタン:

- 自動再生ボタン。
- 字幕ボタン。
- 設定ボタン。
- ミニプレーヤーボタン。
- ピクチャーインピクチャーボタン。
- シアターモードボタン。
- テレビで再生ボタン。
- 全画面ボタン。

設定可能な表示方式:

- 自動: YouTube のデフォルト表示方式を維持します。
- 非表示: 対応するボタンを非表示にします。
- 強制有効: 対応するボタンをできるだけ表示状態に保ちます。

注意事項:

- 字幕ボタンは「自動」と「非表示」のみ対応し、「強制有効」は提供しません。
- YouTube のページ構造が変わると、一部ボタンの表示制御が一時的に無効になる場合があります。

<a name="player-mini-player"></a>

## スクロールミニプレーヤー

<a name="player-mini-player-scroll"></a>

### スクロール時に動画をフローティング再生

有効にすると、動画再生ページで下へスクロールしてプレーヤーから離れたときに、現在の動画がフローティングミニプレーヤーになり、動画を見ながらコメントやおすすめ内容を閲覧しやすくなります。

適した使用場面:

- 動画を見ながらコメント欄も閲覧したい。
- 再生ページを下へスクロールしても、動画を視界から外したくない。
- 長い動画の再生中に、見ながら情報を探したい。

設定可能な項目:

- サイズ: 360×203、420×236、480×270、560×315、640×360、720×405 を選択できます。
- 位置: 左上、上部中央、右上、左下、下部中央、右下を選択できます。
- 画面余白: ミニプレーヤーと画面端の距離を設定します。
- プレーヤーを超えてスクロールしたら有効化: 元のプレーヤーから何ピクセル離れたらミニプレーヤーに入るかを設定します。

注意事項:

- この機能は通常の動画再生ページでのみ動作します。
- 全画面再生中やピクチャーインピクチャー再生中はミニプレーヤーに入りません。
- ミニプレーヤーには閉じるボタンがあります。クリックすると今回のスクロール中はフローティングが閉じられます。プレーヤー付近へ戻るか動画を切り替えると、再び発動できるようになります。
- 画面が小さすぎる場合、実際の表示サイズは画面外にはみ出さないように自動的に縮小されます。
- 小さいサイズのミニプレーヤーでは、表示に適さない一部の操作ボタンが隠されます。

<a name="player-ui"></a>

## インターフェース

<a name="player-ui-ending-overlay"></a>

### 半透明の終了時おすすめ内容

有効にすると、動画の終了時に画面上へ重なるおすすめ動画、作者入口などの要素が半透明になり、遮りを減らします。

適した使用場面:

- 動画終了時のおすすめカードによく画面を隠される。
- 動画最後の数秒の内容をはっきり見たい。

注意事項:

- これらのおすすめ要素にマウスをホバーすると通常表示に戻り、クリックしやすくなります。
- この機能はプレーヤー内の終了時オーバーレイ要素にのみ影響します。

<a name="player-ui-progress-bar"></a>

### 動画下部の進行バー

有効にすると、YouTube プレーヤー操作バーが自動的に隠れたときに、動画下部へ簡潔な進行バーが表示されます。

適した使用場面:

- 完全な操作バーを表示せずに再生進行状況を見たい。
- YouTube 標準の進行バーと操作バーによる遮りを減らしたい。

設定可能な項目:

- 進行バーを表示するかどうか。
- 進行バーの高さ。1 から 20 ピクセルに対応します。

注意事項:

- 進行バーは主にプレーヤー操作バーが隠れているときに表示されます。
- ライブ動画ではこの進行バーは表示されません。

<a name="player-ui-time-tag"></a>

### 角の動画時間

有効にすると、プレーヤー操作バーが自動的に隠れたときに、プレーヤーの角へ現在の再生時間と総時間が表示されます。

適した使用場面:

- 完全な操作バーを開かずに、いつでもどこまで再生したか知りたい。
- 長い動画を見るときに、より目立つ時間表示が必要。

設定可能な項目:

- 動画時間を表示するかどうか。
- フォントサイズ。8 から 48 に対応します。
- 表示位置: 左下、右下、左上、右上。
- 端からのオフセット。0 から 200 ピクセルに対応します。

注意事項:

- 時間バッジは主にプレーヤー操作バーが隠れているときに表示されます。
- ライブ動画では通常、現在の再生時間のみ表示されます。

<a name="player-other"></a>

## その他

<a name="player-other-max-volume"></a>

### 最大音量時に本当の 100% 音量を使用

有効にすると、YouTube の音量バーが 100% のときに、拡張機能は動画要素自体も本当の最大音量状態になるようにします。

適した使用場面:

- YouTube では音量最大と表示されているが、実際の音がまだ小さい。
- プレーヤーが追加で音量を下げていないことを確認したい。

注意事項:

- この機能は YouTube の音量バーが 100% のときにのみ介入します。
- これは追加の音量増幅ではありません。100% を超える音量が必要な場合は「音量ブースター」を使用してください。
- 動作中は音量スライダーが目立つ色で示されます。

<a name="player-other-nonstop"></a>

### 動画の自動一時停止を無効化

有効にすると、拡張機能は長時間再生後の YouTube の自動一時停止通知をできるだけ阻止し、「動画が一時停止されました。再生を続けますか？」のようなダイアログを検出したときに自動処理します。

適した使用場面:

- 音楽、ポッドキャスト、ライブ配信アーカイブ、学習動画を長時間再生する。
- 長時間操作していないために YouTube が一時停止するのを避けたい。

注意事項:

- この機能は YouTube の自動一時停止系の挙動のみを対象とします。
- ユーザーの手動一時停止、ネットワーク問題、ページ制限によって動画が停止した場合、拡張機能は再生継続を保証しません。

<a name="translate"></a>

# 翻訳

この分類の機能は字幕とコメントの翻訳に使用します。字幕翻訳とコメント翻訳はオンライン翻訳サービスを使用するため、ネットワークが利用可能である必要があります。

<a name="translate-settings"></a>

## 翻訳設定

<a name="translate-settings-target-language"></a>

### 翻訳先言語

この設定は、字幕翻訳とコメント翻訳をどの言語へ翻訳するかを決定します。

設定可能な項目:

- YouTube 言語に従う: 現在の YouTube ページ言語を自動的に使用します。
- 言語を指定: 一覧から固定の目標言語を選択します。

注意事項:

- 原文言語と目標言語が同じ場合、一部の翻訳機能は自動翻訳をスキップします。
- 目標言語を変更した後、すでに表示されている古い翻訳内容の更新には、ページの再読み込みやコメントの再読み込みが必要な場合があります。

<a name="translate-settings-never-translate"></a>

### 常に翻訳しない言語

この設定は、特定言語のコメントを自動翻訳しないために使用します。

適した使用場面:

- 直接読める言語があり、それらを自動翻訳したくない。
- 中国語コメント、英語コメント、またはその他の慣れた言語で翻訳文が重複して出るのを避けたい。

注意事項:

- この設定は主にコメントの自動翻訳に影響します。
- コメントが翻訳不要な言語として識別された場合でも、拡張機能は手動翻訳ボタンを残し、ユーザーはクリックして翻訳を確認できます。

<a name="translate-subtitle"></a>

## 字幕翻訳

<a name="translate-subtitle-enable"></a>

### 字幕翻訳を有効化

有効にすると、拡張機能は YouTube が字幕を読み込むときに字幕内容を翻訳し、翻訳文をプレーヤー字幕内に表示します。

適した使用場面:

- 外国語動画を見るときに字幕翻訳が必要。
- 元の字幕を残しつつ翻訳も見たい。
- 字幕を直接目標言語に置き換えたい。

設定可能な項目:

- 字幕翻訳を有効化: 動画字幕を翻訳するかどうかを制御します。
- 翻訳方式: 「バイリンガル翻訳」または「翻訳文のみ表示」を選択できます。

翻訳方式の説明:

- バイリンガル翻訳: 字幕内に翻訳文と原文を同時に表示します。
- 翻訳文のみ表示: 翻訳後の内容のみ表示します。

注意事項:

- この機能には、動画自体に利用可能な字幕が必要です。
- 字幕翻訳を有効または無効にすると、現在の YouTube ページが再読み込みされる場合があります。
- 字幕内容が長すぎる場合や翻訳サービスが失敗した場合、拡張機能は元の字幕を保持します。
- 字幕翻訳はページ読み込み内容を調整する機能です。ページに異常が出た場合は、まずこの機能を無効にして確認できます。

<a name="translate-comment"></a>

## コメント翻訳

<a name="translate-comment-content"></a>

### コメント内容を翻訳

有効にすると、拡張機能はコメント欄内のコメントを自動翻訳し、コメント原文の近くに翻訳文を表示します。

適した使用場面:

- 多言語のコメント欄を閲覧する。
- 海外動画を見るときにコメント内容を素早く理解したい。
- YouTube 標準の翻訳ボタンを 1 件ずつクリックしたくない。

設定可能な項目:

- コメント内容を翻訳: コメントを自動翻訳するかどうかを制御します。
- 翻訳先言語: コメントをどの言語へ翻訳するかを決定します。
- 常に翻訳しない言語: 慣れた言語で翻訳文が自動表示されるのを避けます。

注意事項:

- コメントが空、記号のみ、または翻訳不要と識別された場合、拡張機能は手動翻訳ボタンを表示します。
- コメントの並び順変更、さらにコメントを読み込み、または YouTube によるコメント欄の動的更新後も、拡張機能は新しく出現したコメントを引き続き処理します。
- 翻訳サービスが利用できない場合、コメントは元のまま保持されます。

<a name="translate-comment-line-by-line"></a>

### 翻訳文を原文下に行ごとに表示

有効にすると、複数行の内容を含むコメントについて、拡張機能は各行の翻訳文を対応する原文の下へ配置し、段落全体の翻訳をコメント末尾にまとめないようにします。

適した使用場面:

- 歌詞、段落付き説明、長いコメント、複数行のタイムラインを翻訳する。
- 原文と翻訳文を行ごとに対照して読みたい。

注意事項:

- この機能は「コメント内容を翻訳」が有効であることに依存します。
- コメント自体に改行が含まれる場合にのみ、行ごとの表示が明確になります。通常の 1 行コメントは従来どおり翻訳文を表示します。

# その他

この分類の機能は、YouTube ページレイアウト、コメント表示、広告の干渉、Shorts、ブランド外観、旧版レイアウト、カスタムスタイルの調整に使用します。

<a name="other-home"></a>

## ホーム

<a name="other-home-video-per-row"></a>

### ホームおすすめ動画の 1 行あたり表示数を固定

有効にすると、YouTube ホームのおすすめ動画を 1 行に表示する数を固定できます。

適した使用場面:

- ホームのカードをより密に、またはよりゆったり表示したい。
- YouTube がウィンドウ幅に応じて 1 行あたりの数を自動調整するのが好みでない。
- 大画面でより多くの動画を表示したい。

設定可能な項目:

- スイッチ: 1 行あたりの数を固定するかどうかを制御します。
- 数量: 1 行に表示する動画数を入力します。

注意事項:

- この機能は主にホームのおすすめ動画グリッドに影響します。
- 現在の入力範囲は 1 から 10 を推奨します。範囲外の値は 4 にリセットされます。
- ページ幅が狭すぎる場合、YouTube 自身のレイアウトが実際の表示に影響する可能性があります。

<a name="other-comment"></a>

## コメント

<a name="other-comment-nickname"></a>

### コメント投稿者のニックネームを表示

有効にすると、拡張機能はコメント投稿者のチャンネル名を表示し、元のアカウント名を補助情報として表示しようとします。

適した使用場面:

- コメント投稿者のより完全なチャンネル表示名を見たい。
- YouTube のアカウント名やハンドルだけを見ることに慣れていない。

注意事項:

- 有効化または無効化後、YouTube ページの再読み込みが必要な場合があります。
- この機能は名前を読み取るためにコメント投稿者のチャンネルページへアクセスする必要があるため、読み込み速度はネットワークと YouTube ページの応答に依存します。
- チャンネル情報を読み取れない場合、コメント表示は YouTube の元の状態を維持します。

<a name="other-comment-auto-more"></a>

### 長いコメントを自動展開

有効にすると、長いコメントが自動的に展開され、コメント内の「もっと見る」と「閉じる」ボタンが非表示になります。

適した使用場面:

- 長いコメントや歌詞コメントをよく読む。
- 「もっと見る」を繰り返しクリックしたくない。

注意事項:

- この機能は主にコメント欄に影響します。
- 有効化後、長いコメントはデフォルトで展開され、手動で閉じる入口は非表示になります。

<a name="other-anti-ad"></a>

## 広告ブロック

<a name="other-anti-ad-image"></a>

### 画像/テキスト広告をブロック

有効にすると、拡張機能は YouTube ページ内の画像/テキスト広告、広告枠、一部の Premium プロモーション、ページ広告モジュールを非表示にします。

適した使用場面:

- ホーム、再生ページ、サイドバー内の広告干渉を減らしたい。
- おすすめコンテンツ領域をよりすっきりさせたい。

注意事項:

- これは実験的機能であり、すべての広告をブロックできる保証はありません。
- YouTube のページ構造が変わると、一部の広告が再び表示される可能性があります。

<a name="other-anti-ad-video"></a>

### 動画広告をブロック

有効にすると、拡張機能はプレーヤー内の動画広告を減らそうとし、広告動画を検出したときにできるだけ早くスキップします。

適した使用場面:

- 動画の前後または途中に挿入される動画広告を減らしたい。
- 動画再生をより途切れにくくしたい。

注意事項:

- この機能を有効または無効にすると、現在の YouTube ページが再読み込みされる場合があります。
- これは実験的機能であり、ページ異常を引き起こす可能性や YouTube に検出される可能性があります。
- 一部の場合では、YouTube の「スキップ」ボタンを手動でクリックする必要が残ります。
- YouTube に再生異常が出た場合は、まずこの機能を無効にして確認できます。

<a name="other-anti-ad-merch"></a>

### チャンネル主の商品おすすめをブロック

有効にすると、拡張機能はチャンネル主の商品おすすめ、ショッピングおすすめ欄、チケットおすすめ、プレーヤー内の一部の商品おすすめ入口を非表示にします。

適した使用場面:

- 再生ページで商品棚やショッピングプロモーションを見たくない。
- 動画周辺の商業的なおすすめを減らしたい。

注意事項:

- この機能は関連するページ要素を非表示にするだけで、動画内容自体は変更しません。
- YouTube のページ構造が変わると、一部の商品モジュールを一時的に非表示にできない場合があります。

<a name="other-shorts"></a>

## Shorts（ショート動画）

<a name="other-shorts-home-search"></a>

### ホームと検索結果の Shorts をブロック

有効にすると、ホームおすすめ内の Shorts セクションが非表示になり、検索結果内の Shorts セクションも一緒に非表示になります。

適した使用場面:

- ホームで Shorts のおすすめを見たくない。
- 検索時に通常の動画結果だけを見たい。

注意事項:

- この機能はホームと検索ページ内の Shorts おすすめ領域に影響します。
- YouTube 上の Shorts コンテンツを削除するわけではなく、対応する入口やセクションを非表示にするだけです。

<a name="other-shorts-watch"></a>

### 動画再生ページ内の Shorts おすすめをブロック

有効にすると、動画再生ページのサイドバー、説明欄、または関連おすすめ領域内の Shorts おすすめが非表示になります。

適した使用場面:

- 通常の動画を見ているときに Shorts おすすめで中断されたくない。
- 再生ページのおすすめ領域を通常動画により集中させたい。

<a name="other-shorts-menu"></a>

### サイドバー内の Shorts をブロック

有効にすると、YouTube サイドバー、ミニサイドバー、モバイルウェブ版の下部ナビゲーション内の Shorts 入口が非表示になります。

適した使用場面:

- Shorts を使わないため、入口の干渉を減らしたい。
- Shorts の誤クリックを避けたい。

注意事項:

- この機能は主に入口を非表示にするもので、Shorts リンクを直接開くことには影響しません。
- YouTube の言語やレイアウトによっては、一部の入口を非表示にできない場合があります。

<a name="other-rollback"></a>

## ロールバック

<a name="other-rollback-player-layout"></a>

### 旧版再生画面レイアウトに戻す

有効にすると、拡張機能は YouTube 再生ページの一部レイアウトを旧版スタイルに近づけます。たとえば、狭めの右側おすすめ欄や小さめのおすすめサムネイルを復元します。

適した使用場面:

- 新版再生ページの右側おすすめが広すぎる、またはサムネイルが大きすぎるのが好みでない。
- 再生ページのレイアウトをよりコンパクトにしたい。
- 16:9 ではない動画がプレーヤー領域内で不自然に表示される。

注意事項:

- この機能は主にデスクトップ版動画再生ページの 2 カラムレイアウトに影響します。
- これはページ外観のロールバック調整であり、YouTube のページ改版後は効果が変わる可能性があります。

<a name="other-appearance"></a>

## 外観

<a name="other-appearance-premium-logo"></a>

### Premium ロゴに偽装

有効にすると、左上の通常の YouTube ロゴが YouTube Premium ロゴとして表示されます。

適した使用場面:

- YouTube Premium ロゴのスタイルが好みである。
- ページ上部を Premium アカウントのような外観にしたい。

注意事項:

- この機能はロゴの外観だけを変更します。YouTube Premium の有効化、アカウントの会員状態の変更、Premium 特典の利用を可能にはしません。
- この機能を有効または無効にすると、現在の YouTube ページが再読み込みされます。
- YouTube がページデータ構造を変更した場合、ロゴが一時的に置き換わらないことがあります。

<a name="other-appearance-logo-country-code"></a>

### ロゴの国/地域表示を変更

テキストを入力すると、YouTube ロゴの横にある国または地域を示す表示が、入力した内容に変わります。空欄にすると YouTube のデフォルト表示に戻ります。

適した使用場面:

- ページ左上のブランド表示をカスタマイズしたい。
- デフォルトの国または地域の略称を好みのテキストに変えたい。

設定可能な項目:

- 国/地域表示: `Pirates` などのカスタムテキストを入力できます。空欄の場合は変更しません。

注意事項:

- この機能はロゴの横に表示されるテキストだけを変更します。アカウントの地域、コンテンツの地域、言語、通貨、おすすめの結果、実際の所在地は変更しません。
- 内容を変更または消去すると、開いている YouTube ページが自動的に再読み込みされます。
- このオプションは単独で使用でき、「Premium ロゴに偽装」に依存しません。両方を同時に有効にすることもできます。
- YouTube がページレイアウトを変更した場合、この表示が一時的に表示されないことがあります。

<a name="other-custom-css"></a>

## カスタム CSS

<a name="other-custom-css-inject"></a>

### YouTube にカスタム CSS を注入

有効にすると、自分で書いた CSS スタイルを YouTube ページに適用し、色の調整、要素の非表示、レイアウト細部の変更に使えます。

適した使用場面:

- CSS に詳しく、個人の習慣に合わせて YouTube を微調整したい。
- 特定のページ要素を一時的に非表示にしたい。
- より深い個人向け外観調整を行いたい。

設定可能な項目:

- スイッチ: カスタムスタイルを有効にするかどうかを制御します。
- テキストボックス: YouTube に適用する CSS を入力します。

注意事項:

- テキストボックスがフォーカスを失うと内容が保存され、開いている YouTube ページに適用されます。
- CSS を誤って書くと、YouTube ページの表示が異常になる可能性があります。
- スイッチをオフにすると、カスタムスタイルはページから削除されます。

## 拡張機能設定

<a name="general-config-export"></a>

### 設定をエクスポート

現在の拡張機能設定をテキストとしてエクスポートし、バックアップ、他のブラウザーへの移行、または他のユーザーとの共有に使えます。

設定可能な項目:

### 設定をリセット

拡張機能設定をデフォルト値へ戻せます。

注意事項:

- リセット前に確認プロンプトが表示されます。
- リセット後、パネルは「プレーヤー」ページへ戻ります。
- 通常設定のリセットでは、チャンネル別記憶を特別に消去しません。記憶を移行または上書きする場合は、インポート/エクスポート内の記憶オプションを使用してください。

## 更新とページ通知

<a name="general-update-notice-version"></a>

### 新バージョン通知

ブラウザーが拡張機能の新バージョンを検出すると、一般ページに新バージョン通知が表示されます。

利用可能な操作:

- 更新履歴を見る。
- 今すぐ更新。

注意事項:

- 通常、すべての YouTube タブを閉じると、ブラウザーは拡張機能の更新を自動的に完了します。
- 「今すぐ更新」をクリックすると拡張機能が再読み込みされ、更新完了後に開いている YouTube タブが更新されます。

<a name="general-update-notice-reload-page"></a>

### YouTube ページの再読み込みが必要な通知

拡張機能の更新後、すでに開いている YouTube ページが古い状態をまだ使用している場合があります。このとき、ページ上に再読み込みを求める通知が表示されることがあります。

適した使用場面:

- 拡張機能が更新されたばかりだが、YouTube ページ内の機能が異常に見える。
- ページに拡張機能が更新済みで、再読み込みが必要と表示されている。

注意事項:

- 通知内の再読み込みボタンをクリックすれば、現在のページを更新できます。
- 再読み込み後、ページは新しい拡張機能状態を使用します。

<a name="general-update-notice-compatibility"></a>

### 互換性警告

字幕翻訳、動画広告ブロック、または Premium ロゴへの偽装を有効にしている場合、一般ページに警告表示が出ます。

理由:

- これらの機能は、YouTube がページに読み込む字幕、広告、ブランド情報を直接調整します。
- この調整は YouTube の改版、ネットワーク問題、ブラウザー制限によってページ異常を引き起こす可能性があります。

推奨:

- 動画が再生できない、字幕が異常、ページ読み込みが異常、または Premium の購入、チャンネルメンバーへの参加、ライブ配信への投げ銭などの支払い関連ページに問題が発生した場合は、まず字幕翻訳、動画広告ブロック、Premium ロゴへの偽装を無効にして確認してください。

### 初回インストール案内
