# 更新履歴

[中文](/docs/zh-cn/CHANGELOG.md) | [English](/CHANGELOG.md) | [日本語](/docs/ja/CHANGELOG.md)

すべての Release バージョンの更新記録をこのファイルに記載します。

質問、BUG 報告、機能提案がある場合は、[GitHub Issues ページ](https://github.com/xlch88/YouTubeTweak/issues) から送信してください。

## [1.2.0] - 2026-07-18

### 更新

- **インストールガイド:** Chrome、Edge、Firefox に応じた拡張機能アイコンの案内を表示し、インストール後の操作ガイドを改善しました。
- **設定画面:** カード、スクロールバー、倍速ボタン、設定管理のレイアウトを改善し、設定のリセットを拡張機能内の確認ダイアログに変更しました。

### 追加

- **外観:** 「Premium ロゴに偽装」オプションを追加し、YouTube の上部バーのロゴを Premium スタイルで表示できるようにしました。

### 修正

- **更新履歴:** 読み込みに失敗した場合でも既読として扱われる問題と、ページ切り替え後に再度自動で開く場合がある問題を修正しました。
- **機能説明:** 英語表示で機能説明への入口を開けない問題を修正しました。
- **スクロールミニプレーヤー:** YouTube のアンビエントモードによってプレーヤー画面が暗くなる問題を修正しました。
- **字幕翻訳:** 一部の字幕イベントにテキスト断片が含まれていない場合にエラーが発生する可能性がある問題を修正しました。

## [1.1.9] - 2026-07-03

### 追加

- **更新履歴:** 内蔵更新履歴ポップアップと新バージョン未読通知を追加し、拡張機能内で現在バージョンの変更内容を確認できるようにしました。
- **設定画面:** 機能説明へのクイック入口を追加し、各設定から対応するヘルプドキュメントを直接開けるようにしました。
- **プレーヤーボタン:** プレーヤーボタンの表示モードを追加し、自動、非表示、強制有効の 3 つのオプションに対応しました。

### 修正

- **カスタム CSS:** カスタム CSS エディターがポップアップウィンドウ内でエラーになる場合がある問題を修正しました。

## [1.1.8] - 2026-07-03

### 更新

- **設定画面:** 設定のインポート/エクスポートと更新確認結果の通知表示を改善しました。
- **プレーヤー操作:** 倍速、回転、ミラーなどのカスタム操作時におけるプレーヤー操作バーの表示を改善しました。

### 追加

- **倍速操作:** 倍速スライダーを追加し、ドラッグによる連続速度調整、マウスホイールによる速度調整、カスタムホイールステップに対応しました。
- **画面ズーム:** マウスホイールで動画画面をズームする機能を追加し、拡大後に画面をドラッグして移動できるようにしました。
- **プレーヤー機能ボタン:** 動画スクリーンショットボタンを追加し、現在の動画画面を PNG 画像として保存できるようにしました。
- **プレーヤーボタン:** プレーヤーボタンの表示モードを追加し、自動、非表示、強制有効に対応しました。
- **カスタム CSS:** カスタム CSS エディターを追加し、YouTube にカスタムスタイルを注入できるようにしました。
- **設定管理:** 記憶データのエクスポート/インポートオプションを追加し、設定移行時にチャンネル別の倍速、字幕などの記憶も一緒に移行できるようにしました。

## [1.1.7] - 2026-06-12

### 更新

- **広告ブロック:** YouTube のショッピングおすすめ欄に関連するブロックルールを追加しました。
- **一時停止の無効化:** 「動画が一時停止されました。再生を続けますか？」ダイアログの検出を改善し、他のポップアップを誤って閉じないようにしました。
- **記憶ストレージ:** 500ms 以内の重複書き込みをまとめて処理し、不要なストレージ更新を減らしました。
- **ミニプレーヤー:** ポップアップウィンドウ内のミニプレーヤーサイズ表示を簡素化しました。
- **i18n:** 新版プレーヤーとロールバック設定に関する翻訳を更新しました。
- **依存関係:** WXT/ビルドツールを更新し、旧版ブラウザーグローバル変数の互換コードを削除しました。

### 追加

- **ミニプレーヤー:** スクロールミニプレーヤーを追加し、サイズ、位置、画面余白、トリガー距離の設定に対応しました。
- **音量ブースター:** 実際の音量ブースト機能を追加し、プレーヤー内操作とカスタムブーストレベルに対応しました。
- **コメント翻訳:** 複数行コメントの翻訳を行ごとに表示するオプションを追加しました。
- **プレーヤー UI:** YouTube のミニプレーヤーボタンを非表示にするオプションを追加しました。
- **ロールバック:** 旧版 YouTube 再生画面レイアウトに戻すオプションを追加しました。

### 修正

- **ミニプレーヤー:** ミニプレーヤーモードへの出入り時の重なり順とレイアウト復元を改善しました。
- **ロールバック:** 旧版再生画面レイアウトを有効にした後、16:9 ではない動画が 16:9 の外枠に押し込まれて表示される問題を修正しました。

## [1.1.5] - 2026-04-11

### 更新

- **広告ブロック:** 2 つの新しいブロックルールを追加しました。

### 追加

- **一時停止の無効化:** 一定時間再生した後の自動一時停止を防ぎます（「動画が一時停止されました。視聴を続けますか？」という通知）。

### 修正

## [1.1.4] - 2025-12-03
