## 🥰 この拡張機能について

このブラウザー拡張は **YouTube** ユーザー向けに設計されており、使用体験の最適化、公式デザイン上の不便の修正、上級ユーザー向けのさらなるカスタマイズ機能の提供を目的としています。
プロジェクトは **MIT オープンソースライセンス** を採用しており、ソースコードは完全に公開され透明で、誰でも自由に閲覧、修正、二次開発できます。

私たちは厳粛に約束します:

- **いかなる形の商業化も行いません**。広告、有料アンロック、付加価値サービスはありません。
- **追跡コードを一切組み込みません**。ユーザーデータの収集、分析、アップロードは行いません。
- **いかなるプライバシー情報も取得しません**。すべての設定、キャッシュ、設定情報はローカルブラウザー内にのみ保存されます。
- **ユーザー体験と自由を尊重します**。すべての機能は手動で有効化または無効化でき、完全にユーザー自身が制御します。

私たちの目標は単純です。YouTube を広告やアルゴリズムの遊び場ではなく、純粋な動画視聴体験に戻すことです。

この拡張機能の機能一覧は拡張ストアで確認できます。または、直接インストールして体験できます。
